## Notes d'exécution
- **Personnalise l'expéditeur** : un vrai prénom + signature humaine ouvre mieux qu'un « no‑reply ».
- **Délivrabilité** : domaine vérifié (SPF/DKIM/DMARC) avant le lancement (cf. barrière go/no‑go).
- **Conditionne** la séquence B sur l'événement « 1ᵉʳ PV généré » : si activé → saute B1/B3 (envoie une version « bravo »).
- **A/B test** : teste les objets (cf. T6 du plan). Vise > 35‑45 % d'ouverture sur la waitlist (liste chaude).
- **1 CTA par email.** Tout le reste (liens secondaires) va en bas / dans le PS.
- **Mobile‑first** : 90 % liront sur téléphone — phrases courtes, bouton large.
